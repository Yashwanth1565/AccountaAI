# AccountaAI

## Overview

AccountaAI is an AI-powered meeting intelligence platform that converts meeting recordings into structured insights, action items, and searchable meeting memory.

## Features

- Meeting transcription using Whisper
- AI-generated meeting summaries
- Decision and risk extraction
- Automatic action item detection
- Accountability scoring
- Meeting memory using RAG
- Email reminder scheduling
- Modern Streamlit interface

## Tech Stack

- Python
- Streamlit
- Google Gemini 2.5 Flash
- OpenAI Whisper
- LangChain
- ChromaDB
- SQLite
- APScheduler
- Resend
- Langfuse
